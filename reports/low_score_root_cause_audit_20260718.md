## 证据确定性分级

本报告中的数据并非全部具有相同的确定性。下列标签是阅读和引用报告时的强制口径：

| 标签 | 含义 | 可否直接作为最终结论 |
|---|---|---|
| `confirmed_objective` | 可重复的执行覆盖、生成状态、文件时长、原生 RTF、指定 ASR 后端的 CER/WER、代码测试结果 | 可以，但必须同时注明模型版本、硬件、运行方式及 ASR 版本 |
| `confirmed_code_root_cause` | 修复前后具有受控对照，并有回归测试或端到端复测支持的代码问题 | 可以，结论限定于已复现的代码路径 |
| `supported_model_boundary` | 多组安全参数、随机种子或处理路径对照均未消除，且不同 ASR 后端方向一致的残余问题 | 可以写成“当前模型/MLX 实现边界”，不能写成模型在所有实现和参数下绝对无法改善 |
| `proxy_metric_only` | UTMOS、单一 ASR 对方言的 CER 等存在语言、口音或数据域偏差的代理指标 | 不可以单独判定模型失败 |
| `human_review_required` | 方言正确性、自然度、情感、音色和主观听感需要合格听评者裁决 | 在人工证据补齐前保持未决 |

因此，本报告可确认客观执行数据和已闭环的代码根因；模型短板结论均限定为当前权重、MLX 实现、已测参数范围和本机环境。闽南语内容正确性以及跨语言 UTMOS 不属于已确定的模型失败。

## 已确认并修复的代码问题

| 问题 | 根因 | 修复 | 验证 |
|---|---|---|---|

| MOSS 普通/长文本被截断 | 评估适配器固定 `48/64` token，约只生成 3.84/5.12 秒 | 改为按文本、语言和能力估算 `112–512` token | 修复后 17/17 成功；CER 0 的普通中文和长英文不再截断；中位 RTF 0.8044 |
| Ming Tarkin 克隆复读参考内容 | 长参考文本误启用 prompt-latent 续写通道 | 短参考保留 prompt text；长参考只用说话人嵌入 | CER/WER `0.961/0.9231 → 0/0`，UTMOS `1.2556 → 3.2449`，SIM `0.575 → 0.771`；普通短中文克隆路径保留 |
| IndexTTS2 极端目标时长音质崩坏 | 通过 3.3–6.7 倍后处理拉伸伪造 10/20 秒目标 | 增加最大拉伸比守门；超限时明确返回未应用及原因 | 20 秒目标保留约 2.995 秒自然输出，不再生成低质伪时长音频；项目测试通过 |
| IndexTTS2 RTF 无法取原生耗时 | CLI 只打印保存成功文本 | CLI 输出结构化生成结果 | 单例真实 RTF 2.3427，确认不是单纯外层计时误差 |
| Scenema 已验证参数未固化 | 中英混读和长中文回退到较差默认参数 | 固化中英混读 pace/VC 策略及长中文较优 seed/voice | 中英混读 1.7B CER 0.0571；长中文 UTMOS 2.7515 → 3.1812，CER 0.3452 → 0.2738 |
| VoxCPM2 闽南语被普通话 ASR 判为异常 | 方言读音被普通话转写成不同汉字，CER 不是可靠内容裁决 | 方言 TTS 不再由通用 CER 自动升级为内容异常 | 自动异常已取消，保留人工听评门禁 |
